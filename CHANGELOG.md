- Added the ability to create skin packs from the Change Skin screen, which can be done by pressing the "Create Skin Pack" button
- Added the ability to add skins to skin packs, which can be done by right-clicking on a skin pack and pressing the "Import Skin" button
    - Skin options include:
        - `Skin Name`
        - `Geometry: Wide or Slim`
        - `Cape Texture`
- Added the ability to edit skin packs and skins from the Change Skin screen, which can be done by right-clicking on a skin pack or skin
- Updated elytra texture logic for skins with capes
    - The default elytra texture will now only be used if the cape texture does not contain an elytra
- The "Favorite Skins" skin pack is now always visible
- Separate offsets can now be set for the Paper Doll on the Title and Pause screens
- Added a hover tooltip to the Change Skin button on the Title and Pause screens
